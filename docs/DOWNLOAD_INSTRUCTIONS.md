### Understanding Markdown
- Markdown is plain text with formatting
- Open with any text editor
- Renders beautifully in VS Code, GitHub, etc.

- `# Heading` - Creates headings
- `**bold**` - Bold text
- `*italic*` - Italic text
- `[link](url)` - Links
- `` `code` `` - Inline code
